# Python Programming (PY101) - 2026

This repository contains exercises and notes from working through Launch 
School’s *Programming Foundations with Python: Basics*.

It continues directly from PY100 and focuses on deepening core programming 
ability through structured problem solving, with an emphasis on writing clear, correct, and maintainable Python code.

The goal at this stage is to move from basic syntax familiarity into more confident, deliberate problem decomposition and implementation.

## Focus Areas

* Data structures (lists, dictionaries, tuples, sets)
* Control flow and iteration
* Functions and modular design
* String processing and manipulation
* Input/output and basic program structure
* Debugging and reasoning about code behaviour
* Problem solving using PEDAC-style thinking

## Structure

* Separate directory for each lesson
